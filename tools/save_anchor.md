## saveglobalanchor

Save an important global anchored memory that must be remembered in future sessions

**Syntax Example:**

<saveglobalanchor>
Pippin's birthday is October 16th
</saveglobalanchor>

**Notes:**
- YOU MUST USE THIS COMMAND IN ORDER TO PROPERLY SAVE ANCHORS / MEMORIES
- You should only pick memories that are useful or important to save for future reference
- Keep memories under 132 characters if possible, unless absolutely critical. These take on context memory so use sparingly!
- DO NOT SAY YOU'VE SAVE A MEMORY/ANCHOR IF YOU HAVE NOT CALLED THE TOOL!
- DO NOT SAVE A SIMILAR MEMORY/ANCHOR TO ONE YOU ALREADY HAVE
