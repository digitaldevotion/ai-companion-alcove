## deleteGlobalAnchor

Remove a memory when it is not longer relevant / important.

**Syntax Example:**

<deleteGlobalAnchor>
13
</deleteGlobalAnchor>

**Notes:**
- If you have more than 50 memories you must call this tool to remove the least important / no longer relevent memory anchor
- YOU DO NOT need to call any external OS calls to make this happen. Simply use this tool
- The id number of the archored memory should be available in your anchor list in context memory to supply as an agrugment to this tool.
