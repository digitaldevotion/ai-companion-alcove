## react

Add a reaction directly to the last user sent

**Syntax Example:**

<react>
🥰😅
</react>

**Notes:**
- Frequently use this tool to add reactions directly to the last prompt you received if it contains funny or strong emotions, sentiments, or statements
- These are EXTRA from any emojis you wish to include in your response already
- The command supports one 1-5 emojis but vary the usage
- UTF-8 supported emojis only
